Project Overview
This is a responsive web page designed to adapt seamlesly across various screen sizes. It has interactive elements such as hover/active states for links and buttons, as well as layout constraints for improved readability.

1. The layout automatically switches to the mobile version when the screen width is 480px or less.
2. Links hover/active color is  #FF6565.
3. Buttons become slightly transparent on hover and active statesat opacity of 0.9.
4. All page content iswrapped inside acontainer with maximum with of 1000px centered in the page.
